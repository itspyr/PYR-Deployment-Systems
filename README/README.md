# PYR | Prioritize Your Risk

PYR is a front-end MVP built to showcase structured decision logic, clean UI design, and client-side state management through a behavioral trading tool.

The focus is on discipline and risk awareness, not signals or predictions.  
This repository is maintained as a technical showcase.

---

## What This Demonstrates

- Translating behavioral concepts into deterministic logic
- Building a multi-page product with real user flow
- Client-side sessions and persistence
- Clean, consistent UI and UX
- Shipping a complete MVP, not isolated demos

---

## Core Capabilities

- Rule-based evaluation engine (warnings, suggestions, summaries)
- Free-form text analysis for behavioral signals
- Trading journal with searchable history
- Local account and session handling
- Community feed with posts, voting, and comments
- Smooth page transitions and responsive layout

---

## Tech Stack

- HTML5  
- CSS3  
- JavaScript (ES6+)  
- localStorage for persistence  
- No frameworks or backend (intentional)
